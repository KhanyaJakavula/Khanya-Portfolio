#  Shopping Cart Web Application

##  Project Overview
This is a dynamic shopping cart web application built using JavaScript. It simulates a basic e-commerce experience where users can add products to a cart, update quantities, and view total prices in real time.

---

##  Purpose of the Project
The purpose of this project is to demonstrate core web development skills including:
- JavaScript logic implementation
- DOM manipulation
- User interaction handling
- Basic e-commerce functionality

---

##  Features
- Add products to shopping cart
- Remove items from cart
- Update product quantity
- Automatic total price calculation
- Interactive user interface

---

##  My Role
I was responsible for:
- Developing the full JavaScript functionality
- Implementing cart logic (add, remove, update)
- Handling DOM manipulation
- Structuring the user interface behavior
