#  Login & Registration System

##  Project Overview
This project is a responsive login and registration system built using HTML, CSS, and JavaScript. It allows users to create accounts and log in with basic form validation.

---

##  Purpose of the Project
The goal of this project is to demonstrate understanding of:
- Form creation and handling
- Input validation
- User authentication UI design
- Frontend web development fundamentals

---

##  Features
- User registration form
- User login form
- Input validation (required fields)
- Error message handling
- Responsive design layout

---

##  My Role
I was responsible for:
- Designing the login and registration forms
- Implementing form validation using JavaScript
- Styling the interface using CSS/Bootstrap
- Structuring user interaction flow

---

##  Technologies Used
- HTML5
- CSS3

- JavaScript
